# 🗺️ 百度地图配置指南

## 为什么选择百度地图？

✅ **更常用** - 国内用户最熟悉的地图服务  
✅ **配置简单** - 只需一个 AK (Access Key)  
✅ **功能丰富** - 支持 2D/3D、卫星图、街景  
✅ **免费额度** - 个人开发者完全够用  
✅ **加载快速** - 国内访问速度快  

---

## 📋 配置步骤（5 分钟完成）

### 步骤 1：注册百度地图开发者

1. 访问：**https://lbsyun.baidu.com/**
2. 点击右上角 **"登录"** 或 **"注册"**
3. 使用百度账号登录（没有就注册一个）

---

### 步骤 2：创建应用

1. 登录后，点击顶部菜单 **"控制台"**
2. 在左侧菜单选择 **"应用管理"** → **"我的应用"**
3. 点击蓝色按钮 **"创建应用"**

---

### 步骤 3：填写应用信息

**应用名称**：
```
绿牧农牧官网
```

**应用类型**：
```
选择 "浏览器端"
```

**Referer 白名单**：
```
*
```
> 💡 说明：`*` 表示允许所有域名访问（测试用）  
> 生产环境建议填写：`lvmu0999.github.io`

**点击 "提交" 按钮**

---

### 步骤 4：获取 AK

1. 提交后，在应用列表中找到刚创建的应用
2. 复制 **"AK"** 列的值（一串字母数字组合）
3. 格式类似：`7a8b9c0d1e2f3g4h5i6j7k8l9m0n1o2p`

---

### 步骤 5：替换到代码中

1. 打开 GitHub 仓库：https://github.com/lvmu0999/lvmu-website
2. 找到 `contact.html` 文件，点击编辑
3. 搜索：`YOUR_BAIDU_MAP_AK`
4. 替换为你的 AK（保留引号）

**替换前**：
```javascript
src="https://api.map.baidu.com/api?v=3.0&ak=YOUR_BAIDU_MAP_AK"
```

**替换后**（示例）：
```javascript
src="https://api.map.baidu.com/api?v=3.0&ak=7a8b9c0d1e2f3g4h5i6j7k8l9m0n1o2p"
```

5. 滚动到底部，填写提交信息：
   ```
   配置百度地图 AK 🗺️
   ```
6. 点击 **"Commit changes"**

## ⚠️ 常见问题

### Q1: 地图显示"地图加载中..."
**原因**：AK 未配置或配置错误  
**解决**：检查 contact.html 中的 AK 是否正确复制

### Q2: 地图显示"APP 不存在"
**原因**：AK 复制错误  
**解决**：重新从百度地图控制台复制 AK

### Q3: 地图显示"Referer 校验失败"
**原因**：Referer 白名单限制  
**解决**：在百度地图控制台修改白名单为 `*` 或 `lvmu0999.github.io`

### Q4: 地图位置不准确
**原因**：坐标需要调整  
**解决**：编辑 contact.html，修改 `point` 坐标：
```javascript
var point = new BMap.Point(经度，纬度);
```
